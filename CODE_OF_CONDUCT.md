# Code of Conduct

## Our Commitment

We are committed to making participation in the Exploidus community a respectful and welcoming experience for everyone.

We encourage:

- Respectful communication
- Constructive feedback
- Collaboration
- Professional behavior
- Helping new contributors

## Unacceptable Behavior

Examples include:

- Harassment or personal attacks
- Discrimination
- Hate speech
- Spam
- Trolling
- Sharing private information without permission

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct and may remove comments, issues, pull requests, or contributors who repeatedly violate these guidelines.

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community spaces related to Exploidus

## Contact

If you experience unacceptable behavior, please contact the project maintainer.

Thank you for helping make Exploidus a welcoming open-source project.
