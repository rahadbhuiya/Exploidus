# Exploidus

Exploidus is an open-source operating system developed from scratch in Bangladesh. It is independently designed and implemented without using the Linux, BSD, or any other existing operating system kernel.

The project includes its own kernel, scheduler, virtual memory manager, virtual file system, ExFS filesystem, networking stack, drivers, userspace components, and programming language support. Exploidus is being developed as a complete operating system platform with a focus on performance, security, and extensibility.

The goal of Exploidus is to demonstrate that a modern operating system can be built from the ground up through original engineering and continuous development.
